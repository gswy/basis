# basis
> 基于SprintBoot2.5，Vue3.0的后台基本框架代码。所有带有后台项目均从此处开始。

# 开发说明

## 目录结构
```
basis              
├── admin       # 项目后台vue模板。
└── server      # 项目后端服务端。
```

## 依赖说明
后端项目依赖：
  - java 8
  - SpringBoot 2.5.5
  - jjwt 0.11.2
  - mybatis-plus 3.4.3
  - lombok
  
前端项目依赖：
  - vue 3.2.20
  - vue-router 4.0.2
  - vuex 4.0.2
  - ant-design-vue 2.2.3
  - axios 0.21.1
  - js-cookie 3.0.0
  
## 项目升级说明
项目升级主要插件，可参考maven或者npmjs官方站点的版本信息。框架未引入依赖层级过多的包。
